# Bộ kỹ năng nội dung — SANOTEL / Nội Thất Cho Con (bản cho nhân viên)

Bộ skill này giúp bạn (nhân viên nội dung) dùng **Claude Code** để **sản xuất nội dung đạt chuẩn** cho
Nội Thất Cho Con / SANOTEL, rồi **đưa nội dung vào bảng Lark "Chờ đăng"**. Hệ thống máy chủ của công ty
sẽ **tự động đăng** lên website / Facebook. Bạn chỉ tập trung **viết cho hay, cho đúng chuẩn**.

## ⭐ Nguyên tắc vàng
> **Bạn CHỈ VIẾT → BỎ VÀO BẢNG LARK. KHÔNG tự đăng lên website / Facebook / TikTok / YouTube.**
> Bộ skill này **không chứa và không cần** mật khẩu website hay token nào. Nếu ai đưa bạn mật khẩu/token,
> **đừng dùng, đừng lưu vào bài** — mọi việc đăng do máy chủ tự lo.

## 🚀 Cách dùng
1. Cài **Claude Code** (bản desktop hoặc VS Code) và đăng nhập tài khoản công ty cấp.
2. Mở thư mục này bằng Claude Code.
3. Gõ yêu cầu, ví dụ:
   - "Viết cho tôi 1 bài blog về *giường tầng trẻ em* chuẩn SEO" → skill **viet-blog-noi-dung** chạy.
   - "Viết caption Facebook cho bài này" → skill **caption-facebook**.
   - "Nghiên cứu từ khoá về bàn học chống gù" → skill **research-seo-web**.
4. Viết xong, làm theo mục "Bỏ vào bảng Lark" trong từng skill → đặt trạng thái **"Chờ đăng"**.

## 📚 Danh sách kỹ năng

### Viết & đăng nội dung (viết → Lark)
- **viet-blog-noi-dung** — viết bài blog chuẩn SEO + AEO + AIO + GEO (bài chủ lực cho website).
- **viet-bai-seo-chuan** — viết bài blog chuẩn SEO tổng quát từ brief.
- **caption-facebook** — viết caption Facebook Hook-Story-Offer kéo traffic về web.
- **hmh-mkt-hook-video** — viết hook "chống lướt" cho video ngắn TikTok/Reels/Shorts.
- **hmh-mkt-content-da-kenh** — hệ thống nội dung đa nền tảng (lịch + tái sử dụng nội dung).
- **hmh-mkt-content-30-ngay** — xây chuỗi nội dung 30 ngày.
- **hmh-mkt-infographic-html** — thiết kế infographic bằng HTML/CSS rồi xuất ảnh.
- **hmh-mkt-ebook-sach-lat** — biến một chủ đề thành ebook hoàn chỉnh có thương hiệu.

### Nghiên cứu & chiến lược (làm brief, nền tảng cho bài hay)
- **hmh-mkt-research-seo-web** — nghiên cứu từ khoá + chủ đề.
